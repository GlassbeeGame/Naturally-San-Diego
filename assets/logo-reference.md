## Logo Variants Needed for Site

Based on the brand guide, the site will need:

1. **Primary logo (header)**: White version for use on dark/colored nav bar — `NSD_WhiteLogo.png` or `NSD_WhiteNaturallyLogo.png`
2. **Favicon**: Leaf icon from `NN_Icon.png` — will need to be converted to .ico or sized appropriately
3. **Footer logo**: Could use the white logo or a green-on-white variant if available
4. **Open Graph / social sharing image**: May need to create from existing assets

## Notes

- The .ai file is the vector source — if we need a color logo on white background, it can be exported from this file
- Per brand guidelines: do NOT alter logos, enclose in shapes, add shadows, or modify the relationship between "Naturally" and "San Diego"
- White knockout versions allow the background to show through the leaf shapes
